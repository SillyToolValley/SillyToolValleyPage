01 / about

Developer tools for the editors you live in.

Silly Tool Valley is a one-person studio building production-grade developer tools: the kind of small, sharp utilities that fix what kept slowing down real projects.

The current catalog starts with Unity editor workflow, real terminal sessions, AI CLI integration, and MCP-aware automation. Every product listed here is designed to survive daily use before it asks for anyone else's time.

  • Daily-driven Every tool earns its place by saving time in real projects first.
  • Editor-first Built around where game developers already work: Unity, shells, source control, and AI assistants.
  • Honest docs Clear setup, clear trade-offs, and no mystery features hidden behind marketing fog.
  • Quietly maintained Focused releases, practical fixes, and tooling that keeps its promises.
