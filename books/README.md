# Books Directory

Each published book gets its own folder under `books/<book-slug>/`.

Expected layout:

```text
books/
  <book-slug>/
    index.html
    reader.html
    assets/
    en/
      <book-slug>.html
      <book-slug>.pdf
    ko/
    ja/
    zh-Hans/
    es/
```

New books should be added only under `books/<book-slug>/`.
